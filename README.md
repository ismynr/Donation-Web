# Cara Install

## Clone Dari Github
-   Masuk terminal / git bash
-   git clone [url_github]
-   cd [nama_repo]
-   composer install
-   cp .env.example .env
-   setting email konfigurasi di .env
-   php artisan key:generate
-   php artisan migrate:fresh --seed

## Perbaharui Repo Lokal Dr Remote
-   git pull [nama_remote] [nama_branch]
-   php artisan migrate:fresh --seed
